## UV Package Manager

### Basic Commands
- `uv init .` - Initialize project in current directory
- `uv add <package>` - Add a dependency
- `uv sync` - Install all dependencies
- `uv run python script.py` - Run your Python code

### Key Difference
- `uv init` → Creates new subdirectory
- `uv init .` → Initializes in current directory
